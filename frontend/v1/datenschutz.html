Datenschutzerklärung

BIW Pokal ist ein privates, nicht-kommerzielles Hobby-Projekt ohne Vereinsstatus, ohne Werbung und ohne Monetarisierung.

Verantwortliche Stelle

Kontakt ausschließlich über den Discord-Server.

Erhebung und Verarbeitung von Daten

Diese Website erhebt keine personenbezogenen Daten. Es werden weder Tracking-Tools noch Analyse-Software eingesetzt. Es gibt keine Cookies.

Lokaler Browserspeicher (localStorage)

Die Website speichert ausschließlich folgende Daten lokal im Browser des Nutzers (localStorage). Diese Daten verlassen niemals den Browser und werden nicht an den Server übertragen:

  • biw_dark_mode – Farbschema-Einstellung (hell/dunkel)
  • biw_token – Temporärer Login-Token für Administratoren

Diese Daten können jederzeit im Browser gelöscht werden (Einstellungen → Browserdaten löschen).

Server-Logs

Der Webserver speichert beim Aufruf der Seite technische Zugriffsdaten (IP-Adresse, Zeitstempel, aufgerufene URL). Diese Daten werden ausschließlich zur Fehleranalyse genutzt und nicht an Dritte weitergegeben.

Externe Links

Diese Website enthält Links zum Discord-Server des BIW Pokal. Beim Klick auf einen solchen Link verlässt du diese Website. Für die Datenschutzpraktiken von Discord ist deren eigene Datenschutzerklärung maßgeblich.

Rechte der betroffenen Personen

Da keine personenbezogenen Daten gespeichert oder verarbeitet werden, gibt es keine Daten, die auskunftspflichtig, löschbar oder übertragbar wären. Bei Fragen wende dich über den Discord-Server an uns.

Änderungen dieser Erklärung

Diese Datenschutzerklärung kann bei Bedarf aktualisiert werden. Der aktuelle Stand ist jederzeit auf dieser Seite abrufbar.
