### Task 3: KO bracket scroll hints

**Files:**
- Modify: `frontend/ko.html:26` (inline CSS, add .ko-scroll-wrapper)
- Modify: `frontend/ko.html:270` (JS, wrap bracket in wrapper div)
- Modify: `frontend/archiv.html:37` (inline CSS, add .ko-scroll-wrapper)
- Modify: `frontend/archiv.html:308,383` (JS, renderCompactBracket/V2 output)

- [ ] **Step 1: Add .ko-scroll-wrapper CSS to ko.html**

In `frontend/ko.html`, find the inline `<style>` block. After the `.ko-bracket` rule (line 26), add:

```css
    .ko-scroll-wrapper{position:relative}
    .ko-scroll-wrapper::after{content:'';position:absolute;top:0;right:0;width:40px;height:100%;background:linear-gradient(to right,transparent,var(--bg));pointer-events:none}
```

- [ ] **Step 2: Wrap bracket output in ko.html**

In `frontend/ko.html`, find line 269-270:
```js
  contentDiv.innerHTML = '';
  contentDiv.appendChild(renderBracket(bracketsCache.brackets[activeTab]));
```

Replace with:
```js
  contentDiv.innerHTML = '';
  const scrollWrapper = document.createElement('div');
  scrollWrapper.className = 'ko-scroll-wrapper';
  scrollWrapper.appendChild(renderBracket(bracketsCache.brackets[activeTab]));
  contentDiv.appendChild(scrollWrapper);
```

- [ ] **Step 3: Add .ko-scroll-wrapper CSS to archiv.html**

In `frontend/archiv.html`, find the inline `<style>` block. After the `.ko-bracket-compact` rule (line 37), add:

```css
    .ko-scroll-wrapper{position:relative}
    .ko-scroll-wrapper::after{content:'';position:absolute;top:0;right:0;width:40px;height:100%;background:linear-gradient(to right,transparent,var(--bg));pointer-events:none}
```

- [ ] **Step 4: Wrap bracket output in archiv.html render functions**

In `frontend/archiv.html`, find `renderCompactBracket()` (line 308):
```js
  let html = '<div class="ko-bracket-compact">';
```

Replace with:
```js
  let html = '<div class="ko-scroll-wrapper"><div class="ko-bracket-compact">';
```

Then find the closing of that function — look for where the function returns `html`. The `</div>` closing `ko-bracket-compact` needs a second `</div>` for the wrapper. Find the line that closes the bracket div (before `return html`):

In `renderCompactBracket()`, find line 371 (immediately before `return html;`):
```js
  html += '</div>';
  return html;
```

Replace with:
```js
  html += '</div></div>';  // close ko-bracket-compact + ko-scroll-wrapper
  return html;
```

Repeat the same pattern for `renderCompactBracketV2()` — find line 383:
```js
  let html = '<div class="ko-bracket-compact">';
```

Replace with:
```js
  let html = '<div class="ko-scroll-wrapper"><div class="ko-bracket-compact">';
```

And find line 454 (immediately before `return html;`):
```js
  html += '</div>';
  return html;
```

Replace with:
```js
  html += '</div></div>';  // close ko-bracket-compact + ko-scroll-wrapper
```
